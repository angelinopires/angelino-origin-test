* TypeScript
* SCSS
* Jest, Enzyme
* Parcel ✨

## 🤖 Prerequisites
* Node 14 or higher
* NPM 7 or higher

## 🦾 How to run locally
After cloning the project:
```bash
$ cd angelinopires/angelino-origin-test
$ npm install
```
To run it locally:
```bash
$ npm start
```
To run unit tests:
```bash
$ npm run test
```
